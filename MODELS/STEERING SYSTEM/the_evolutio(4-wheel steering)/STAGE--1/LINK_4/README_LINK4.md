# LINK4

## Description
This folder contains the CAD model of Link 4 used in the four-wheel steering mechanism.

## Function
Link 4 transfers steering movement between steering components while helping maintain steering synchronization between wheels.

## Features
- Strong mechanical design
- Lightweight construction
- 3D printable
- Easy integration into the steering system

## Files Included
- LINK4.f3d
- LINK4.step
- LINK4.stl

## Notes
Part of the steering linkage system used for coordinated wheel steering.
