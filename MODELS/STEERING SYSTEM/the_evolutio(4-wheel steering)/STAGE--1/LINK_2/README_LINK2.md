# LINK2

## Description
This folder contains the CAD model of Link 2 used in the four-wheel steering mechanism.

## Function
Link 2 transfers steering motion between adjacent steering joints while maintaining the required steering geometry. It ensures smooth and synchronized movement.

## Features
- High structural strength
- Lightweight design
- Easy to manufacture using 3D printing
- Supports accurate steering movement

## Files Included
- LINK2.f3d
- LINK2.step
- LINK2.stl

## Notes
Used together with the remaining steering links for proper steering operation.
