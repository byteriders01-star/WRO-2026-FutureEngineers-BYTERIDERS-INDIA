# Four-Wheel Steering Mechanism

## Overview

This folder contains the CAD models of the four-wheel steering linkage used in the WRO Future Engineers robot. The steering mechanism is designed to provide precise steering control by transmitting motion from the steering actuator to all four wheels through a series of mechanical links.

The design improves maneuverability, reduces turning radius, and ensures synchronized steering movement for autonomous navigation.

## Steering Linkage Components

| Link | Description |
-----------------------------------------------:
| LINK1 | Primary steering link that transfers motion from the steering actuator to the linkage system. |
| LINK2 | Connects steering joints and continues steering motion through the mechanism. |
| LINK3 | Intermediate steering link that maintains steering geometry and synchronization. |
| LINK4 | Transfers steering movement between linkage assemblies. |
| LINK5 | Connects steering components while improving steering stability and accuracy. |
| LINK6 | Final steering link that completes the four-wheel steering assembly. |

## Purpose

The steering linkage was developed for the **World Robot Olympiad (WRO) Future Engineers** category. These components are part of the robot's mechanical steering system and are intended to provide reliable and repeatable steering performance during autonomous driving.
