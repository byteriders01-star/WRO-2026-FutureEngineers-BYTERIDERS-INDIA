# LINK6

## Description
This folder contains the CAD model of Link 6 used in the four-wheel steering mechanism.

## Function
Link 6 serves as the final connecting linkage in the steering assembly, ensuring coordinated steering motion and maintaining mechanical stability.

## Features
- Strong and reliable design
- Lightweight
- Easy to manufacture using 3D printing
- Compatible with the complete steering assembly

## Files Included
- LINK6.f3d
- LINK6.step
- LINK6.stl

## Notes
This component should be assembled with the complete steering linkage system for proper four-wheel steering operation.
