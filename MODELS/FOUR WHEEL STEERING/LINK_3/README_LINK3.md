# LINK3

## Description
This folder contains the CAD model of Link 3 used in the four-wheel steering mechanism.

## Function
Link 3 connects steering joints and transmits steering input while maintaining proper alignment between connected components.

## Features
- Compact design
- Smooth steering motion
- Optimized for strength and weight
- Suitable for rapid prototyping

## Files Included
- LINK3.f3d
- LINK3.step
- LINK3.stl

## Notes
Designed specifically for the steering linkage assembly.
