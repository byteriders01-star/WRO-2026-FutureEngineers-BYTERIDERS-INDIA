# LINK1

## Description
This folder contains the CAD model of Link 1 used in the four-wheel steering mechanism of the WRO Future Engineers robot.

## Function
Link 1 is the primary steering linkage that transfers motion from the steering actuator to the steering assembly. It plays an important role in synchronizing wheel movement and maintaining steering accuracy.

## Features
- Lightweight mechanical link
- Optimized for precise steering transmission
- Compatible with the four-wheel steering assembly
- Designed for 3D printing

## Files Included
- LINK1.f3d – Native Autodesk Fusion 360 model
- LINK1.step – Universal CAD file for compatibility
- LINK1.stl – STL file for 3D printing

## Note:
This component is part of the complete four-wheel steering mechanism and should be assembled with the remaining steering links.
