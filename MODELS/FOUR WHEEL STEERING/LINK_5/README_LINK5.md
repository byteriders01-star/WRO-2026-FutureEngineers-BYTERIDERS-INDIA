# LINK5

## Description
This folder contains the CAD model of Link 5 used in the four-wheel steering mechanism.

## Function
Link 5 connects steering elements and transfers steering force between mechanical joints. It contributes to accurate steering response and overall system stability.

## Features
- Durable linkage design
- Lightweight structure
- Optimized for 3D printing
- Supports precise steering motion

## Files Included
- LINK5.f3d
- LINK5.step
- LINK5.stl

## Notes
Works together with the other steering links to complete the steering mechanism.
